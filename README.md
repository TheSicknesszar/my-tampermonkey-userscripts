# 🎯 Arena.ai Lightbox Enhanced + Batch Download

A Tampermonkey userscript that adds a full-featured lightbox to [arena.ai](https://arena.ai), [lmarena.ai](https://lmarena.ai), and [chat.lmsys.org](https://chat.lmsys.org) with original/edited image detection, EXIF metadata, zoom/pan, touch swipe, and batch download.

## Features

- **Lightbox**: Click any image to open it in a full-screen overlay with navigation
- **Original/Edited Detection**: Analyzes filenames and EXIF data to classify images
- **EXIF Metadata**: Displays camera make/model, dates, software, and more
- **Zoom & Pan**: Ctrl+/Ctrl- to zoom, drag to pan, R to rotate
- **Touch Support**: Swipe left/right on mobile
- **Batch Download**: Download all images at once with a progress overlay
- **Selection Mode**: Choose specific images before batch downloading
- **Keyboard Shortcuts**: Full keyboard navigation (press `H` for help)

## Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/) for your browser
2. Open the [raw script](https://raw.githubusercontent.com/TheSicknesszar/my-tampermonkey-userscripts/main/Arena-AI_Lightbox_Enhanced.js)
3. Tampermonkey will prompt you to install — click **Install**

## Supported Sites

- `https://arena.ai/*`
- `https://chat.lmsys.org/*`
- `https://lmarena.ai/*`
- `https://www.lmarena.ai/*`
